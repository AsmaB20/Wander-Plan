# ✦ WanderPlan

Link: [https://wander-plan-bice.vercel.app/](https://wander-plan-bice.vercel.app/)

AI-powered trip planner — days, hotels, places, costs, and Gemini-powered recommendations.

## Stack

- **Frontend**: Vanilla HTML/CSS/JS + jsPDF (client-side PDF generation)
- **Backend**: Express.js (Node.js)
- **AI**: Google Gemini 2.0 Flash
- **Deploy**: Vercel (single serverless function routing)

---

## Local Development

```bash
npm install
npm run dev        # → http://localhost:3000
```
